# NixOS configuration file

## Installing NixOS

Follow the instructions on these two pages:
* https://guides.frame.work/Guide/NixOS+on+the+Framework+Laptop+13/400
* https://mynixos.com/nixos-hardware

In the NixOS config file, you need:
<nixos-hardware/framework/13-inch/amd-ai-300-series>

## Installation

Run the `install` command, which creates a symbolic link to this directory containing the important `.nix` files.
