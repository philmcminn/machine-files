# Setting up a new macOS machine

1. Copy the setup_ssh script to a USB stick and run it in the terminal 
2. Clone the config repository.
3. Install [Brew](https://brew.sh/). Ignore the message at the end of the install requesting further configuration changes.
4. Install `.zshrc` by going to the `zshrc` directory and running `install`.
5. Go to the `brew` directory and type `brew bundle --verbose`.
